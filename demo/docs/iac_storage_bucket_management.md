# IaC Management for Storage Bucket

The Google Cloud Storage bucket `driftscribe-hack-2026-tofu-artifacts` in the `asia-northeast1` region is now managed under Infrastructure as Code (IaC).

This change ensures that the bucket's configuration is version-controlled and deployed consistently, reducing configuration drift and improving operational reliability.

For general guidance on configuring Cloud Storage with Cloud Run services, refer to the official documentation: [Configure Cloud Storage volume mounts for Cloud Run services](https://docs.cloud.google.com/run/docs/configuring/services/cloud-storage-volume-mounts).
